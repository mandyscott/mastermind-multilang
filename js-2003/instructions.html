Object: Guess the secret code!

Instructions: Type in your choice of four coloured pegs to see if you can guess the secret code.
Possible peg colours are: red, yellow, green, blue, white, or black
You have 16 chances to guess the secret code, otherwise the game is over.
The code maker will add black and white pegs to the board to help you decide which pegs to try next.
Refresh the page to start a new game (you will lose the current game if you are half-way through!).

Results: The black and white pegs from the code maker represent the following:
* A black peg is placed in the result for each peg in your guess which is the same colour and in the same place as a peg in the secret code.
* A white peg is placed in the result code for each peg in your guess which is of the correct colour in an incorrect position.
* Black pegs are always displayed before white pegs in the result.
